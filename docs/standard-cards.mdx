# Standard Trick Cards

GitWarp evaluates your pasted GitHub URL to determine its context (User, Repo, File, Commit, or PR). It then activates the appropriate "Trick Cards" below. 

## Development & Execution
* **bolt.new**: Imports the repository into an AI builder agent standing by to edit & ship.
* **stackblitz.com**: Boots the repository in an interactive browser WebContainers dev environment.
* **codespaces.new**: Instantly launches a GitHub Codespaces cloud environment.
* **gitpod.io**: Launches an automated cloud development environment on Gitpod.

## Editing & Navigation
* **github.dev**: Opens the repository or file in a full VS Code web editor.
* **github1s.com**: Classic VS Code web view for fast read-only codebase navigation.
* **vscode.dev**: Opens the repository or file in VS Code for Web without extension setup.

## AI & Analysis
* **deepwiki.com**: Auto-generates a Wikipedia-style wiki and interactive Q&A assistant.
* **gitdiagram.com**: Reads the repository and draws an interactive architecture diagram.
* **gitingest**: Flattens repo or file into one prompt-friendly text document with token count.
* **gitmcp.io**: Converts the repository into a live MCP server for AI coding assistants.
* **gitpodcast.com**: Generates an audio podcast with host dialogue explaining the codebase.
* **github.gg**: Repository control panel: one-click copy for AI, security scan, and quality score.
* **star-history.com**: Interactive star growth chart over time for evaluating repository health.

## Native GitHub Tricks
* **.keys**: Fetches all public SSH keys associated with the GitHub user account.
* **.gpg**: Fetches all public GPG signing keys associated with the GitHub user account.
* **.patch**: Appends `.patch` to commit or PR URLs to get formatted raw Git patch files.
* **.diff**: Appends `.diff` to commit or PR URLs to get raw unified diff output.
* **releases.atom**: Atom RSS feed tracking all published releases and tags.
* **commits.atom**: Atom RSS feed tracking commit history for the repository or branch.

## Raw Data
* **SSH Clone URL**: Formats repository location into a standard Git SSH clone URL.
* **Zip Archive**: Direct link to download a ZIP archive of the repository codebase.
* **Raw File URL**: Direct link to fetch raw file contents from `raw.githubusercontent.com`.
